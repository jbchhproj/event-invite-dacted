\# Product Brief

\#\# Overview  
A modern event invitation and RSVP platform designed to reduce friction for guests while giving hosts better control and visibility.

\#\# Problem  
Existing platforms:  
\- Require RSVP before viewing event details  
\- Create unnecessary friction (accounts, logins)  
\- Poorly handle group/party invitations  
\- Lack visibility into who is attending

\#\# Solution  
\- Frictionless guest access (no account required)  
\- Party-based invitation model  
\- Multi-event support under one invitation  
\- Transparent RSVP system (guests can see who is attending)

\#\# Target Users  
\- Event hosts (weddings, parties, gatherings)  
\- Guests invited to events

\#\# Core Value  
\- Simple for guests  
\- Structured and powerful for hosts
